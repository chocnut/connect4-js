## Connect4 - JavaScript/TypeScript [![Build Status](https://travis-ci.org/chocnut/connect4-js.svg?branch=master)](https://travis-ci.org/chocnut/connect4-js)

## Requirements
- NodeJS

## Usage
```sh
$ cd connect4-js
$ yarn
$ yarn build
```

## Development mode
```sh
$ yarn dev
```

## Testing
```sh
$ yarn test
```
